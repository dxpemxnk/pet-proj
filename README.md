# pet-proj

🗺️ Roadmap
Ближайшие задачи
Настроить Vite для клиента (сборка, dev-сервер, прокси на API)
Страница создания книги — форма (название, автор, страницы, категория)
Поиск книг на клиенте (подключить к существующему API)
Редактирование книги — форма с предзаполненными данными
Отзывчивый дизайн — адаптация под мобильные устройства
Уведомления — snackbar/alert при успешных и ошибочных действиях
Среднесрочные задачи
Пагинация / бесконечный скролл для списка книг
Сортировка и фильтрация (по категории, по автору, по страницам)
Профиль пользователя — смена пароля, аватар
Управление категориями из UI (сейчас есть только API)
Подтверждение email при регистрации
Тесты: юнит-тесты для сервисов, интеграционные для API
Долгосрочные задачи
Docker — контейнеризация приложения
CI/CD — автоматическая проверка и деплой
Избранное / закладки на книги
Комментарии и рейтинги к книгам
Роли и права — администратор, модератор
Логирование — structured logging (Winston/Pino)
Документация API — Swagger/OpenAPI
